New Quest

Quest

You are about to create a new scavenger hunt. Here are some tips for creating your quest:

First you will be taken to the map interface where you have to place your first marker:

  • Start task: This marker defines the location where the players can find your quest in the world.

Important tools:

  • Save the quest: Saves your changes
  • Cancel editing: Dismisses your changes
  • Edit quest: Edit the quest's content
  • Edit storyline: Edit the storyline to create advanced quests.
  • Add rewards: Add rewards the user can win after completing the quest

Tasks

Now add some more tasks to your quest by clicking :

  • Information task: This marker is used to convey useful information to the user
  • Quiz task: This marker challenges the user with multiple choice and gap-fill tasks
  • Search task: The user has to find a hidden marker
  • Fight task: The user has to fight an enemy in a trading card game

For simple linear quests you can change the order of the tasks by dragging them up or down .

More options for each task are available in the task menu next to each task :

  • Delete: Remove the task from the quest
  • Edit: Edit the task's content
  • Preview: Gives you an idea of how the task will look in the app
  • Focus: Focus the task's marker on the map

Now you should be prepared to create your scavenger hunt. Have fun!
